# serde-tuplex-derive

Procedural macros for [serde-tuplex](https://crates.io/crates/serde-tuplex).

This crate provides `#[derive(Tuple)]`, `#[derive(Lenient)]`, and `#[derive(TupleLenient)]`.

## Usage

Do not use this crate directly. Use `serde-tuplex` instead:

```toml
[dependencies]
serde-tuplex = "0.1"
```

See the [main documentation](https://docs.rs/serde-tuplex) for usage examples.
